# biblioteca-senai
Atividade que consiste em criar um sistema bibliotecário em Python usando Flask.

REQUISITOS
1 - Cadastro de Usuários
2 - Autenticação de Usuários
3 - Segurança de Senhas
4 - Cadastro de Livros
5 - Validação de Dados
6 - Registro de Saída (Empréstimo)
7 - Regra de Negócio por Perfil (Prazos)
8 - Regra de Negócio por Perfil (Limites)
9 - Movimentação de Estoque
10 - Consulta ao Acervo
11 - Visualização de Detalhes
12 - Painel do Aluno
13 - Painel do Professor
14 - Relatório Estatístico
